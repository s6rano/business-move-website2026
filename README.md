# Business Move Website 2026

Static trilingual website for finding corporate movers in Belgium by postcode and distance radius.

## Features

- FR / NL / EN interface
- Light and dark modes
- Local mover database
- Postcode and radius search
- Indicative quote calculator
- Email draft with three mover contacts

## Local Preview

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173/index.html`.
